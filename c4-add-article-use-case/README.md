## Installation

1. Open the terminal up on this directory and run:
````
npm install
````
2. Add a file to the root of this directory called ```.env``` and add the following:
````
VUE_APP_API_URL=http://example-api4.glitch.me/api/
````
3. Run:
````
npm run serve
````
